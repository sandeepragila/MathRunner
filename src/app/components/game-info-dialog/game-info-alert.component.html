Math Runner

Welcome to the game! The game tests your math computing ability. You will be shown a grid and imagine you are on the first cell(top left) and your princess is in the last cell (bottom right). You want to get there with as fewer steps as possible. But, you need to follow some simple rules below

  • Each block represents number of miles.
  • Last block (princess) is where you need to land at the end by covering the exact number of miles displayed above the grid.
  • You can only go to the right, down or right+down (diagonal) at any point.
  • You can come back in the same way to undo a step.
  • When you go right or down you ADD the total miles you have travelled so far with the miles on that block.
  • When you go right+down which is diagonal then you MULTIPLY total miles you have travelled so far with the miles on that block.
  • Remember there will be a lot of ways but we want the shortest path (using fewer number of blocks).

Go get your princess!
