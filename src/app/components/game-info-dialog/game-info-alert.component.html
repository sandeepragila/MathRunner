Math Runner

Welcome to the game! The game tests your math computing ability. You will be shown a grid and imagine you are on the first cell(top left) and your princess is in the last cell (bottom right). You want to get there with as fewer steps as possible. But, you need to follow some simple rules below
  • Each block represents a number that you can think the number of miles it takes to complete the block.
  • You can only go to the right, down or right+down(diagonal) at any point
  • You can come back in the same way to undo a step
  • When you go right or down you add the number of miles on that block
  • When you go right down which is diagonal then you multiply the miles on that block
  • Remember there will be lot of ways but we want the shortest path.

Go get your princess!
